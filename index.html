Blacksburg Transit bus tracker with live map updates of all available routes

View project on GitHub

Blacksburg Transit bus tracker with live map updates for all the available Blacksburg Transit routes.

Favorite stops and receive notifications when you are at your favorited stops. Get departure times for any stop. See where the bus is and how many passengers are on it.

Not affiliated with Blacksburg Transit.

To join the beta group visit https://play.google.com/apps/testing/com.bt4vt
